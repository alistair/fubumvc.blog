<h1>FubuMVC.Blog</h1>
Sample fubu mvc project using:

* FubuMVC
* SparkViewEngine
* RavenDB
* JavaScript (require, underscore, amplify, showdown, and jQuery)

<h2>Documentation:</h2>
To run: click on: \packages\RavenDB\Start.cmd open FubuMVC.Blog.sln and click F5.

<h1>FubuMVC.Blog</h1>
A blogging web application using:

* FubuMVC
* SparkViewEngine
* mongoDB
* JavaScript (require, underscore, amplify, showdown, and jQuery)

<h2>Documentation:</h2>
To run:
* Run 'rake', to pull packages and build the project.
* Run mongoDB
* Run FubuMVC.Blog.sln, and click F5 start the project.
